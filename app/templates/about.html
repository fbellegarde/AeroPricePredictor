About Our ML & Generative AI

The **AeroPricePredictor** uses a **Content-Based Recommender System** trained on historical flight attributes (destination, time of year, airline) to predict future price trends.

Generative AI (specifically a **Markov Chain**) is used to create dynamic, flight-themed slogans for a more engaging user experience.

This entire system is built and deployed using modern **Cloud Architecture (AWS/GCP/Azure via Terraform)**, showcasing a robust, full-stack application pipeline.
